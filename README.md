# Media List App

A simple React app that allows users to view, add, and delete items from **Books**, **Movies**, and **Music** lists. Each category is displayed separately, and items can be managed dynamically.

---

## Features

- Display **Books, Movies, and Music** in separate lists  
- **Add new items** dynamically via input fields  
- **Delete items** individually  
- Uses React **state management** and **.map()** for rendering lists
